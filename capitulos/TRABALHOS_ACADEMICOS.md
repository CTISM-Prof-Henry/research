# Tipos de Trabalhos Acadêmicos

|                                                                   | Documento       | Descrição                                                                                                                     |
|:-----------------------------------------------------------------:|:----------------|:------------------------------------------------------------------------------------------------------------------------------|
| <img src="imagens/tipos_trabalhos/fichamento.jpg" width="300px">  | **Fichamento**  | Registro resumido de um texto ou obra, destacando suas principais ideias e informações relevantes.                            |
|   <img src="imagens/tipos_trabalhos/sinopse.jpg" width="300px">   | **Sinopse**     | Breve resumo que apresenta a essência de um conteúdo, geralmente usado em obras literárias, filmes ou peças teatrais.         |
|   <img src="imagens/tipos_trabalhos/resenha.jpg" width="300px">   | **Resenha**     | Análise crítica de uma obra, como livros, filmes ou artigos, com comentários sobre seu conteúdo, estilo e relevância.         |
|  <img src="imagens/tipos_trabalhos/relatório.jpg" width="300px">  | **Relatório**   | Documento que apresenta dados e informações sobre um determinado assunto, resultado de pesquisas ou atividades.               |
|   <img src="imagens/tipos_trabalhos/projeto.png" width="300px">   | **Projeto**     | Documento que descreve um plano detalhado para a realização de uma atividade, incluindo objetivos, metodologias e cronograma. |
|  <img src="imagens/tipos_trabalhos/memorial.jpg" width="300px">   | **Memorial**    | Narrativa autobiográfica ou de eventos históricos, geralmente destacando experiências e realizações significativas.           |
|   <img src="imagens/tipos_trabalhos/resumo.jpg" width="300px">    | **Resumo**      | Síntese das principais informações de um texto, destacando os pontos mais importantes de forma concisa.                       |
|   <img src="imagens/tipos_trabalhos/poster.png" width="300px">    | **Pôster**      | Apresentação visual de informações sobre um tema, frequentemente usada em conferências e eventos acadêmicos.                  |
|    <img src="imagens/tipos_trabalhos/paper.jpg" width="300px">    | **Artigo**      | Trabalho acadêmico ou científico que aborda um tema específico, geralmente publicado em revistas ou periódicos.               |
| <img src="imagens/tipos_trabalhos/monografia.png" width="300px">  | **Monografia**  | Estudo aprofundado sobre um tema específico, geralmente apresentado como requisito para a conclusão de um curso de graduação. |
| <img src="imagens/tipos_trabalhos/dissertação.png" width="300px"> | **Dissertação** | Trabalho acadêmico que apresenta uma pesquisa original, geralmente exigido para a obtenção do grau de mestre.                 |
|    <img src="imagens/tipos_trabalhos/tese.png" width="300px">     | **Tese**        | Trabalho acadêmico que apresenta uma pesquisa original e aprofundada, geralmente exigido para a obtenção do grau de doutor.   |
